for문

for(초기식; 조건식; 제어식)
{

}

1. 초기식 초기화가 먼저 되는 것이 1번 
2. 조건식 판별 안으로 들어갈지 판별한다. 실행 되면 코드 실행
3. 더이상 코드 실행이 될 것이 없다면 제어식으로 와서 count가 1 증가하게 된다.
4. 다시 조건식을 판별한다. 

초기식은 처음만 본다. 



for문에서 
for( ; ; )
{
	// 1

	// 2

	// 3

}

1번째 부분이 만족하면 더이상 실행 시키지 않게 해도 되는 그런 것

break 와 continue


while (조건식)
{
	//반복문에서 빠져나가 주세요
	break;

	명령문
}


2중 while이였을때 


while (조건식1)
{
	while(조건식2)
	{
		break;

		명령문
	}

}

위처럼 되면 조건식 1은 실행되는 조건이였을 때 조건식 1은 실행되고 

break가 담긴 조건식 2번을 빠져나가게 된다. 

범위에 찝혀있는 그 범위만 나가게 된다. 
필요 없어서 강제로 빠져나가는


해당 조건을 간략히 통과하고 싶은 경우 
continue을 사용한다. 

예를들어 

while (조건식)
{

	명령문1

	continue;
	명령문2
}

이경우 명령문 1이 실행되고 continue를 만나면서 명령문2 를 건너뛰고 while문이 다시 실행된다.
